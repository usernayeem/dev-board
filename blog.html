Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select elements in the DOM using JavaScript: 1.By Id, 2.By Class, 3.By Tag Name, 4.By Query Selector

Question-2: What is the difference between innerHTML, innerText and TextContent?

innerHTML retrieves or modifies an element's HTML contents, including text, tags, and attributes. innerText retrieves or modifies an element's visible text content while taking CSS styles. TextContent retrieves or modifies an element's entire text content, excluding HTML tags and including hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM is a pattern used to handle events efficiently by attaching a single event listener to a parent element instead of individual child elements

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on a child element propagates upward through the DOM tree, triggering event handlers on each ancestor until it reaches the root of the document.

Question-5: How do you create, add and remove elements in Javascript?

In JavaScript to create a new element, we can use the document.createElement() method, To add a element, we can use the appendChild() method, To remove a element, we can use the remove() method
